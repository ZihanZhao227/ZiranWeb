# ZiranLab

Personal portfolio and creative blog by Zihan Zhao.

**Live site → [ziranlab.com](https://ziranlab.com)**

## What's here

- **About** — who I am and what I care about
- **Work** — projects I've built
- **Everything Bagel** — a shelf of writing in three flavors:
  - *Everyday* — ordinary thoughts
  - *Inedible* — ideas that might not sit well with everyone
  - *Medley* — things made for the internet

## Stack

Next.js · TypeScript · Tailwind CSS · Notion API · Framer Motion · Vercel

## Notes

Content is managed through Notion and pulled via API at build time.
The site is a work in progress — more projects and writing coming.
